# ScriptNet

# Un potente script Python pensato per automatizzare e gestire operazioni di network scanning, test di sicurezza ed amministrazione di sistema direttamente da riga di comando (CLI).

Caratteristiche principali:

1. Installazione Automatica: Configura e installa in autonomia dipendenze di base e tool avanzati di sicurezza tramite un'unica opzione.
2. Menu Interattivi Multipli: Organizzato in sottomenu intuitivi per scansioni Nmap, attacchi DoS, gestione Wi-Fi (Aircrack-ng), generazione di payload (MsfVenom) e analisi di metadati.
3. Gestione dei Servizi di Rete: Controllo rapido di interfacce, NetworkManager e routing tramite Tor.
4. Prompt Dinamico e Personalizzabile: Interfaccia grafica da terminale pulita, con rotazione casuale del prefisso (ScriptNetwork, SN, SNPython), colorazione ANSI e supporto completo in lingua inglese.
